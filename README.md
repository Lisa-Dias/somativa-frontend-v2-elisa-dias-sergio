# Code & Coffee
Este projeto está sendo desenvolvido para a somativa 1 da matéria de front-end, para ter como objetivo do usuário conseguir interagir com nosso site Code & Coffee. Ele poderá ver as bebidas do dia e escolher uma delas, caso ele queira despertar, ou manter acordado, ou se não tiver interesse, poderá cancelar tudo.

## 🛠️ Tecnologias Utilizadas
* HTML5 (Semântica e Acessibilidade)
* CSS3 (Estilização e Feedback Visual)
* JavaScript (Manipulação de DOM e Eventos)

## 🚀 Funcionalidades Implementadas
* **Interatividade Visual:** Elementos para quando o usuário passar o mouse irá ver as recomendações do dia.
* **Saudação Dinâmica:** Mensagem de bom-dia para quando o usuário chegar antes do 12h,  mensagem de boa-tarde para quando ele estiver entre os horários de 12h e 18h, e uma mensagem de boa-noite para se ele chegar após 18h.
* **Cálculo em Tempo Real:** Formulário para quando o usuário colocar a quantidade de bebidas que quer pedir, e o site já mostra o preço enquanto vai acrescentando a quantidade que quer.
* **Injeção de Conteúdo:** O usuário poderá digitar o nome do café que tem disponível nas recomendações.
